# HiPick Landing Page

AI-powered live streaming highlight clipping landing page built with React, Vite, Tailwind CSS, and Recharts.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deployment

This project is configured for GitHub Pages via GitHub Actions.

- Production build output: `dist`
- GitHub Pages base path: `/3k-landing/`
- Deployment workflow: `.github/workflows/deploy.yml`
